# Программа ToDoManager

- Современное desktop приложение для управления задачами, созданное с использованием Qt6, QML и PostgreSQL.

## Возможности

 **Управление задачами** — создание, редактирование, удаление и завершение задач
- **Система тегов** — организация задач с помощью цветных тегов
- **Статистика** — отслеживание активных, завершенных и отмененных задач
- **Фильтрация** — по статусу, тегам и пользователям
- **Приоритеты** — три уровня приоритета задач (низкий, средний, высокий)
- **Сроки выполнения** — установка и отслеживание дедлайнов
- **Современный интерфейс** — интуитивный UI на QML

---

## Требования

Для запуска приложения необходимы:

- **Qt** 6.5 или выше
- **PostgreSQL** 13 или выше
- **CMake** 3.16 или выше
- **Qt Creator** (рекомендуется)
- Компилятор с поддержкой **C++17**

## Использование

### Основные шаги:

1. **Запустите приложение** и выберите пользователя из выпадающего списка
2. **Создайте задачу** — нажмите кнопку `Задача`
3. **Заполните данные**:
   - Заголовок (обязательно)
   - Описание (опционально)
   - Срок выполнения в формате `ГГГГ-ММ-ДДTЧЧ:ММ` (например: `2025-01-20T15:00`)
   - Приоритет (низкий, средний, высокий)
   - Статус (активная, завершенная, отмененная)
   - Теги (можно выбрать несколько)
4. **Управляйте задачами**:
   - Редактируйте — изменение данных задачи
   - Завершайте — автоматическая установка статуса "завершена"
   - Удаляйте — полное удаление задачи
5. **Используйте фильтры** в левой панели для быстрого поиска
6. **Отслеживайте статистику** — активные, завершенные, общее количество

### Горячие клавиши:

- `Ctrl+R` — обновить данные
- `Esc` — закрыть диалоговое окно

## Технологии

Проект построен с использованием:

- **Qt 6.5+** — фреймворк для создания кроссплатформенных приложений
- **QML** — декларативный язык для создания UI
- **C++17** — основной язык программирования
- **PostgreSQL** — реляционная база данных
- **CMake** — система сборки проекта
